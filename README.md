## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session
- morgan

## Dev Dependencies

- Mocha
- Chai
- nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command, or `npm start` for nodemon command.
